# 📊 Machine Learning & Deep Learning Lab Assignments

This repository contains a collection of **Machine Learning and Deep Learning experiments** implemented as part of my academic coursework. The experiments focus on building practical understanding of core ML/DL algorithms using real-world datasets and modern frameworks.

---

## 🚀 About the Repository

* 📚 Covers fundamental to advanced ML/DL concepts
* 🧠 Hands-on implementation using Python
* 📊 Includes model training, evaluation, and visualization
* 🛠️ Tools & Libraries: `NumPy`, `Pandas`, `Matplotlib`, `Scikit-learn`, `TensorFlow`, `Keras`

### ✅ Completed Experiments

1. ✔️ **Linear & Logistic Regression**

   * Implemented regression models on real-world datasets
   * Evaluated using accuracy, precision, recall, and RMSE

2. ✔️ **Multiple Regression, Lasso & Ridge Regression**

   * Explored regularization techniques
   * Compared bias-variance tradeoff

3. ✔️ **Decision Tree & Random Forest**

   * Built classification models
   * Compared performance and overfitting behavior

4. ✔️ **K-Nearest Neighbors (KNN)**

   * Implemented KNN algorithm
   * Evaluated using different values of K

5. ✔️ **Support Vector Machine (SVM)**

   * Applied SVM for classification
   * Performed hyperparameter tuning (kernel, C, gamma)

6. ✔️ **K-Means & Hierarchical Clustering**

   * Performed unsupervised learning
   * Visualized clusters using plots

7. ✔️ **Artificial Neural Network (ANN)** ⭐ *(Compulsory)*

   * Built ANN using Keras/TensorFlow
